# NoScroll

> *A small extension to remove distracting elements from Instagram*


## Current Features:
When enabled, NoScroll:
- Hide Home page stories
- Hide Reels sidebar button
- Hide Explore page sidebar button
- ~~Hide Home page feed~~ (Temporarily Disabled)
- Redirect ```/reels``` to the Home page
- Redirect ```/explore``` to the Home page
